<h1>Mobile Chapter Manager</h1>

<h2>Description</h2>

 - This Mobile Chapter Manager is an Android-based mobile application that allows FBLA (Future Business Leaders of America) officers easily manage their local chapters.
 - The Android application consists of various pages (each displaying useful information via text fields, calendar views, redirecting buttons, and webviews) all accessible via a hideable sidebar menu.

<h2>Languages and Utilities Used</h2>

- <b>Java</b>
- <b>XML</b>

<h2>Environments Used </h2>

- <b>Anroid Studio</b>
